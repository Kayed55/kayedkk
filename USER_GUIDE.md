# دليل المستخدم — نظام الجودة للتقييم والتدريب (شركة محزم)

دليل موجز لاستخدام النظام. الواجهة عربية بالكامل (RTL) وتعمل على الجوال والتابلت والكمبيوتر.

---

## تسجيل الدخول
1. أدخل **البريد الإلكتروني** و**كلمة المرور**.
2. يصلك **رمز تحقّق (OTP)** من 6 أرقام على بريدك — أدخله خلال 5 دقائق (3 محاولات).
3. نسيت كلمة المرور؟ اضغط **"نسيت كلمة المرور؟"** فتصلك كلمة مؤقتة، غيّرها بعد الدخول من **الملف الشخصي**.

## الأقسام (القائمة الجانبية)
على الجوال اضغط زر القائمة **☰** أعلى اليمين لفتحها.

| القسم | الوظيفة |
|---|---|
| **لوحة التحكم** | ملخّص الأداء: عدد التقييمات، المتوسطات، الناجح/الراسب، أحدث النشاط. |
| **التقييمات** | عرض كل التقييمات، البحث، عرض/تعديل/حذف (حسب الصلاحية). |
| **تقييم جديد** | إنشاء تقييم لموظف: اختيار الموظف، تعبئة البنود، تظهر النتيجة مباشرةً. |
| **الموظفون** | بيانات الموظفين وسجلّ تقييمات كل موظف ومتوسطه. |
| **الاعتراضات** | تقديم/متابعة الاعتراضات على التقييمات والبتّ فيها. |
| **التقارير** | تقرير الأداء الشامل + رسوم بيانية + تصدير Excel/PDF. |
| **التقرير الشهري** | أداء الموظفين لشهر محدّد مع التصدير. |
| **تقرير الإجراءات / الأخطاء** | الإجراءات المتّخذة والأخطاء المتكرّرة. |
| **سجل العمليات** | تدقيق كامل لكل عملية (من قام بها ومتى). |
| **إدارة المستخدمين** | إضافة/تعديل المستخدمين وأدوارهم وإعادة تعيين كلمات المرور (إدمن). |
| **الإعدادات** | معايير التقييم وإعدادات النظام (إدمن). |
| **الملف الشخصي** | تغيير كلمة المرور وبيانات الحساب. |

## قاعدة النجاح والرسوب
- **85% فأكثر = ناجح** ✅
- **84% فأقل = راسب** ❌

## التقييم النهائي والحذف
- عند **حذف تقييم** يُحذف نهائياً من كل الأقسام والتقارير ولا يمكن التراجع (يظهر تأكيد أحمر). الحذف متاح للمدير وموظف الجودة فقط.
- البيانات تتحدّث تلقائياً عند التنقّل بين الأقسام — لا حاجة لتحديث الصفحة.

## الأدوار والصلاحيات
- **مدير (admin):** كل الصلاحيات.
- **موظف الجودة:** التقييمات، الاعتراضات، التقارير، إدارة المستخدمين.
- **مشرف:** عرض فريقه وتقاريرهم واتخاذ إجراءات.
- **موظف:** عرض تقييماته وتقديم اعتراض.

## نصائح
- الإشعارات في الجرس 🔔 أعلى الصفحة.
- كل عملية تُظهر رسالة تأكيد (Toast) بالأعلى.
- لأمانك: لا تشارك رمز الدخول، وغيّر كلمة المرور المؤقتة فور الدخول.
